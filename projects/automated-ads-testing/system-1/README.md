## Insights
- Messaging first, audiences second so we are not dependent on audience features.
- If messaging and targeting are done right, managing ads should be easy.

- Large audience is the way
  - Target a large audience so that you can up scale without running into ad fatigue.
  - Start with a broad audience you absolutely know your prospect is in. If you don't convert, you know it's a messaging issue and not a targeting issue.
  - Start broad and hone in on the best audience, or we can update our messaging so that more people are pulled into it too.

- "Begin with the end in mind"
- Tests are run for the duration of a day
- If we do coded configuration of google ads we can do abo and cbo more easily

## Vision

- Follow the copywriting dossier to hone in on winner messaging
- Generate ads + landing pages based on the messaging
- Run automated ads testing (Variable isolation) to improve over control (Winning ad formula)
- Report on what makes ads successful by analyzing results from the winning ad formula
